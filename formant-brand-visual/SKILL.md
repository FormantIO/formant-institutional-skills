# Formant Brand Visual Language

Desaturated cinematic landscapes. High-contrast command tones. The Generative Horizon represents the vast, untapped potential of the Physical AI frontier — where data is the primary light source, and every image implies structure, precision, and architectural stability.

## The 4 Visual Pillars

### Command Aesthetic
Deep Command (`#0A0F11`) base simulates a mission-control environment where data is the primary light source. All visuals start from darkness and are illuminated by the system.

### Generative Horizon
Desaturated cinematic landscapes representing THE BRAIN, THE BODY, THE CONTEXT — the vast, untapped Physical AI frontier. Mountain ranges, fog, and twilight as recurring motifs.

### Systemic Grid
System Neutral (`#A0A8AB`) lines and grids imply structure, precision, and architectural stability throughout the visual language. The grid is felt, not decorative.

### Functional Glow
High-vibrancy accents (Interface Iris `#8B8CF4`, Terminal Amber `#E8A87F`) used sparingly to guide the eye toward critical human oversight moments. Light is informational, not ornamental.

## Photography & Video Direction

### Atmosphere
- Desaturated, cinematic, fog-heavy
- Twilight or low-light conditions
- Deep shadows with selective illumination
- Landscapes imply frontier, scale, unknown territory

### Subjects
- Solitary figures in vast environments — human scale against operational scale
- Command workstations glowing against dark, foggy landscapes
- Mountain ranges, lakes, fog, mist as environmental metaphors
- Curved command rooms with monitor arrays

### Color Treatment
- Desaturate toward the brand palette (Deep Command, Fog Green, System Neutral)
- Selective color pops from Terminal Amber (warm light sources) and Interface Iris (AI/system glow)
- Never bright, never saturated, never "friendly" — sophisticated mystery

### Still Image Categories

**Atmospheric & Cinematic:**
- Solitary figures by misty landscapes with amber light
- Mountain peaks with signal lines in Terminal Amber on Deep Command
- Foggy thresholds, archways, paths through mist

**Operations & Command:**
- Illuminated command workstations against dark foggy mountain landscapes
- Glowing monitors in dense coastal fog with amber reflections
- Remote command stations with amber glow in vast terrain
- Operators at workstations against geometric/pyramid mountain forms

**Product & UI:**
- The Interface shown as data visualization screens in architectural space
- UI screenshots with the brand's dark treatment
- AI message interfaces synthesizing operational data

## Generative Image Master Prompt

Use this as a base when generating brand imagery:

> "Cinematic wide-angle shot of a desaturated, misty mountain range at twilight, representing the unknown territory of the AI frontier. The atmosphere is thick with fog and mystery. In the foreground, a minimalist, high-tech command console glows with Interface Iris (#8b8cf4) and Terminal Amber (#FFB800) light. The aesthetic is 'Systems Architect' — sophisticated, dark, and architectural. No clutter. The light source feels like a steady guide in a vast expanse. 8k resolution, Unreal Engine 5 render style."

Adapt this prompt by swapping the foreground subject and environmental context while maintaining:
- Desaturated cinematic tone
- Fog/mist atmosphere
- The brand color accents as light sources
- "Systems Architect" aesthetic
- No clutter — minimal, precise, atmospheric

## Video Direction

- Hero videos: atmospheric loops, slow movement, fog and light
- Background loops: abstract, desaturated, minimal motion
- UI demos: dark interface, terminal-style overlays
- All video should feel like a mission-control observation — calm, steady, ambient

## What to Avoid

- Bright, saturated colors or "friendly" aesthetics
- Stock photography feel — generic office, generic tech
- Busy compositions with too many visual elements
- Literal robot/machinery imagery without atmospheric treatment
- Anything that feels like traditional SaaS marketing visuals
